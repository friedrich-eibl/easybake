# EasyBake
A blender plugin to make texture baking simple.

## Installation
- Download the repository and save it to a directory of your choice
- In Blender install via: **Edit -> Preferences -> Add-ons -> Install**
- Activate "Procedural Texture Baker" in the Add-ons list
- Find "Bake" in the side Panel of your 3D viewport

## Usage
You can choose an object you want to bake the textures for and which maps to create (Diffuse / Roughness / Normal). You can also tune the Render Settings and file format to your liking. If selected it will even automatically create and assign the new baked material based on the baked textures.
